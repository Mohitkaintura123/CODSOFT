## CodSoft Machine Learning Internship

This repository contains the projects completed during my Machine Learning Internship at CodSoft.

The internship focused on applying machine learning and deep learning techniques to solve real-world problems such as fraud detection, customer churn prediction, and text generation.

During this internship I worked on data preprocessing, feature engineering, model training, and evaluation using modern machine learning tools.

---

## Projects Completed

1. Credit Card Fraud Detection

This project focuses on detecting fraudulent credit card transactions using machine learning techniques.

## Key Features

- Data preprocessing and exploratory data analysis
- Handling highly imbalanced fraud datasets
- Feature engineering for transaction behavior
- Model training using XGBoost
- Threshold optimization using Precision-Recall Curve

## Results

Precision : 0.84
Recall : 0.74
F1 Score : 0.78

The model successfully identifies fraudulent transactions while minimizing false alarms.

## Technologies used:

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn

---

2. Customer Churn Prediction

This project predicts whether a customer will leave a service using machine learning models trained on customer banking data.

## Key Features

- Data preprocessing and feature scaling
- Handling class imbalance using SMOTE
- Training multiple models
- Hyperparameter tuning
- Model explainability using SHAP

## Results

Accuracy : 85%
F1 Score : 64%
ROC-AUC : 0.87

The model helps businesses identify customers likely to churn so they can take preventive action.

## Technologies used:

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- Matplotlib

---

3. Handwritten Text Generation using LSTM

This project implements a character-level LSTM neural network to generate text similar to human writing.

The dataset was built by collecting research papers and extracting text from PDFs.

## Key Features

- Automated dataset creation from research papers
- Character-level language modeling
- LSTM based deep learning architecture
- Text generation based on seed prompts
- Interactive interface using Gradio

## Technologies used:

- Python
- TensorFlow / Keras
- NumPy
- PyPDF
- Requests
- Matplotlib
- Gradio
